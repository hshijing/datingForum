## 获取所有板块
- 接口地址 /board/loadBoard
  
- 请求参数
  无

- 返回
  ```json
    {
        "status":"success",
        "code":200,
        "info":"请求成功",
        "data":[
            {
                "boardId":10002,//板块ID
                "pBoardId":0,//父板块ID
                "boardName":"Easybbs开发",//板块名称
                "cover":null,//封面
                "boardDesc":"Easybbs开发",//简介
                "sort":1,//排序
                "postType":1,//是否可以发帖 0:不可以 1:可以
                "children":[//子版块
                    {
                        "boardId":10005,
                        "pBoardId":10002,
                        "boardName":"技术问题",
                        "cover":null,
                        "boardDesc":null,
                        "sort":1,
                        "postType":1,
                        "children":[

                        ]
                    },
                    {
                        "boardId":10017,
                        "pBoardId":10002,
                        "boardName":"bug反馈",
                        "cover":null,
                        "boardDesc":null,
                        "sort":2,
                        "postType":1,
                        "children":[

                        ]
                    }
                ]
            }
        ]
    }
  ```
